# Hello World! - 엑셀시트 관련 처리 라이브러리 

## xlwings

홈페이지 : http://xlwings.org/

### 설치방법

#### Windows일 경우

	pip install xlwings
	pip install pypiwin32

자세한 내용은 http://docs.xlwings.org/en/stable/installation.html 
사이트를 참조하시면 됩니다.

#### Mac일 경우

	아직 안 해 보았음.

### 실행방법

	cd xlwings
	python hello.py
